# REACT SETUP

## Installation Setup

1. `npm create vite@latest .`
2. Vite options:
   - React
   - Typescript + React Compiler
   - No (Vite 8 beta)
   - Yes (Install with npm and start)

## Execution Setup

1. Start project: `npm run dev`
2. Stop project: `ctrl + c`

## Additional Command

1. `npm install` / `npm i`: Menginstall package yang terdaftar di file `package.json`

## NOTES

- `npm i [PACKAGE_NAME]` / `npm install [PACKAGE_NAME]` / `npm i --save-dev [PACKAGE_NAME]`
